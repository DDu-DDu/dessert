# 🧁 선주문 관리 시스템 (Dessert Preorder System)

---

## 🛠️ 기술 스택
- Java 17
- Spring Boot
- Thymeleaf
- Gradle

---

## 🚀 실행 방법

1. IntelliJ에서 프로젝트 열기  
2. Gradle 프로젝트로 import  
3. DessertApplication 실행
4. http://localhost:8080 접속

---

## 💡 느낀 점

이번 과제를 통해 단순 기능 구현이 아닌,  
👉 실제 비즈니스 상황에서 발생할 수 있는 문제를  
👉 코드로 어떻게 풀어낼지 고민하는 경험을 할 수 있었습니다.

앞으로도 사용자와 서비스 흐름을 고려한 개발을 지향하겠습니다.
